# KSK
# 2 DKV KSK
Projek tugasan untuk kursus DKV 3263 (2 DKV KSK).
## PENERANGAN
Repo ini mengandungi kod sumber dan dokumentasi berkaitan projek KSK yang dibangunkan bagi memenuhi keperluan kursus.
## TUJUAN
- Memenuhi keperluan tugasan kursus DKV 3263.
- Mengaplikasikan kemahiran pembangunan sistem dan pengaturcaraan.
- Meningkatkan pemahaman terhadap penggunaan teknologi web.
- Menghasilkan projek yang berfungsi dan mesra pengguna.
- Memberi pengalaman praktikal dalam pembangunan dan dokumentasi projek.
## TEKNOLOGI DIGUNAKAN
- HTML
- CSS
- JavaScript
## CARA PENGGUNAAN
1. Muat turun atau clone repository ini.
2. Buka fail `index.html` menggunakan pelayar web.
3. Gunakan sistem seperti yang diarahkan.
## OBJEKTIF PROJEK
- Membangunkan sistem yang memenuhi keperluan pengguna.
- Mengaplikasikan kemahiran pembangunan web.
- Menghasilkan dokumentasi projek yang lengkap.
## PENUTUP
- Projek ini dibangunkan sebagai tugasan kursus DKV 3263. Diharapkan projek ini dapat membantu meningkatkan pemahaman terhadap pembangunan sistem dan penggunaan teknologi web.
